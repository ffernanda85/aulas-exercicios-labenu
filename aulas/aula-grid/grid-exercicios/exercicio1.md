# Exercício 1

Utilize `grid-template` na barra de navegação e na área de galeria, com as seguintes instruções:

  - Barra de navegação:
    - apenas uma coluna;
    - 3 linhas, sendo as duas últimas com 40px, e a primeira com todo o espaço restante
    
- Galeria:
  - 3 colunas
  - 8 linhas de 100px cada
